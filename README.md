# Aladdin AI — Institutional-Grade Trading Intelligence

Next-generation algorithmic trading platform for NSE (Indian Markets), 
Crypto, and Forex. Built on a Hybrid Intelligence System combining a 
Universal Transformer foundation model with RAG-based news sentiment.

## What makes it different

Traditional trading bots react to technical indicators. Aladdin understands 
market context — price action + news sentiment combined into a single 
confidence score before any trade signal is issued.

## Architecture
Market Data (Price/Volume)          Financial News
↓                                  ↓
Time-Series Transformer          MiniLM Vector Embeddings
(Multi-Head Attention)           RAG Semantic Search
↓                                  ↓
Price Prediction              Sentiment Score (Bullish/Bearish)
↓                                  ↓
Hybrid Confidence Score
↓
Trade Signal / No-Trade Filter
## AI Core

- **Model** — Multi-Head Attention Transformer optimized for time series, 
  trained on 100+ assets
- **Universal Foundation Model** — single model predicts Stocks, Crypto, 
  and Forex without per-asset retraining
- **Semantic Engine** — MiniLM embeddings + RAG pipeline reads live financial 
  news and blocks trades during high-risk news cycles
- **Inputs** — quantitative (OHLCV) + qualitative (news sentiment) fused 
  at inference time

## Features

- **Live Dashboard** — real-time prices, interactive charts (Lightweight Charts), 
  live AI signals
- **Strategy Backtester** — simulate strategies on historical data (2022–2024) 
  before live deployment
- **Smart Wallet** — simulated trading account with automatic monthly capital 
  injection and portfolio tracking
- **Daily Briefings** — automated pre-market analysis and post-market accuracy 
  reports generated by AI
